# slim-company

> 这是我用PHP的slim3微框架写的一个企业官网项目，为了更好地SEO优化，没有做前后端的分离，用的是php的twig模板，和前端JS模板引擎很像。其中项目中有用到

* PHP(7.x)
* slim
* MYSQL
* twig
* monolog
* Amazeui
* tinymce
